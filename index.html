Улучшение вашей системы

  • Зарегистрируйтесь
    на GitHub

  • Сделайте Fork моего
    репозитария

  • Скачайте GitHub App
    для OS X или Windows

  • Подключите в Github App
    ваш Fork

  • Создайте свой бранч
    с уникальным именем

Потратьте немного времени и научитесь использовать силу GitHub

Когда ваш код окажется там, будет очень просто его комментировать и править.

Подготовка к работе

  • Создайте файл
    index.html

  • Подключите файл
    reset.css

  • Подключите файл
    style.css

  • Подключите Roboto
    в Google Font

  • Подключите Font
    Awesome

Теперь всё готово для вёрстки

Вспомните всё чему вы научились на курсе и сверстайте этот шаблон.

Сдача работы

  • В GitHub App
    сделайте Commit

  • Синхронизируйте
    проект в GitHub

  • Отправьте Pull
    Request из браузера

Поздравляем со сдачей работы

Наконец-то ваш проект на GitHub. Скоро вы получите комментарии по коду.
